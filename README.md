# 🚀 CampusPulse – Anonymous Student Feedback Platform

CampusPulse is a Reddit/Quora-inspired anonymous platform that enables students to share honest feedback about campus life while providing administrators with real-time, data-driven insights.

---

## 🌐 Overview

CampusPulse bridges the gap between students and campus administration by transforming scattered complaints into structured, actionable insights.

### 🎯 Key Goals
- Enable anonymous and honest student feedback
- Identify trending and critical campus issues
- Provide real-time analytics to administrators
- Create a transparent feedback loop

---

## ✨ Features

### 👤 Anonymous User System
- No login required for students
- Auto-generated unique user ID (e.g., `User_45821`)
- Secure tracking for moderation

### 📝 Post Creation
- Title, description, category selection
- Optional image upload
- Categories:
  - 🍽 Food
  - 🏫 Facilities
  - 🚨 Safety
  - 📚 Academics
  - 🧹 Cleanliness

### 👍 Voting System
- Upvote / Downvote posts & comments
- Trending and priority ranking

### 💬 Comment System
- Nested replies (1 level)
- Reactions: 👍 😡 😂
- Top comments prioritized

### 🚨 Reporting & Moderation
- Report posts/comments
- Auto moderation (threshold-based)
- User penalty system (ban logic)

### 🌍 Translation Feature
- One-click translation for posts/comments
- Multi-language support

### 🧠 Smart Issue Detection
- Sentiment analysis (AI-powered)
- Detect recurring complaints
- Generate:
  - 🔥 Trending Issues
  - 🚨 Critical Alerts

### 🧑‍💼 Admin Dashboard
- Analytics & insights
- Moderation panel
- Issue tracking (Open / In Progress / Resolved)
- Admin responses visible to students

### 🔄 Transparent Feedback Loop
- Students see issue status
- Admin responses publicly visible
